# ARTI406 – Assignment 2

This project demonstrates basic **data preprocessing and dimensionality reduction** techniques using the Iris dataset. The dataset contains 150 records of iris flowers with four numerical features describing sepal and petal measurements, along with a species label.

In this assignment, the data was inspected for quality issues, missing values were handled using mean imputation, and outliers were detected using the IQR method. The numerical features were then normalized using both **Min-Max scaling** and **Z-score standardization**. Finally, **Principal Component Analysis (PCA)** was applied to analyze the explained variance and understand how dimensionality reduction can simplify the dataset while preserving important information.
